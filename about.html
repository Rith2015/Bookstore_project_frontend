About Inkspire Library

Inkspire Library is an intuitive and user-friendly platform designed to make managing and accessing a collection of books effortless for both admins and library users. The frontend consists of several key pages that provide a smooth and dynamic user experience, allowing seamless navigation and interaction with the library's offerings.

Client-side Overview

Inkspire Library client-side is built with HTML and styled with Bootstrap for a responsive and modern design. It includes essential features such as a search bar for quick book discovery, a carousel that showcases featured books, and a series of forms for managing customers and loans.

Pages and Features:

navbar and offcanvas - In every page

The navbar and offcanvas is in every page. The navbar lets the users search for books by name or auther, and also log in as admin . The nav bar system has the following elements:

  • Search Bar: Easily search for books by title or author.
  • offcanvas Bar: Provides access to all sections of the site.
  • Admin Login: Allows administrators to securely log in through a modal.

The offcanvas is a way for users to switch between every page in the site

index.html - Homepage

The homepage is the entry point for all users. It introduces the library system with the following elements:

  • Carousel: Highlights featured books dynamically to attract user attention.

all_books.html - Explore the Collection

The All Books page displays the entire library in an organized, grid-style layout:

  • Responsive Design: Utilizes Bootstrap to ensure the books are presented in a clean, responsive grid.
  • Book Cards: Each book is displayed as a card, showing its title, author, image, and availability status.
  • Book Details and Loan Initiation: Clicking a loan button allows users to begin a loan process, taking them to the loan form.

new_customer.html - Add New Customers

This page allows new customers to be added to the library system:

  • Customer Form: A straightforward form that captures necessary information about a new customer.
  • Seamless Integration: Once submitted, the customer is added to the system, enabling them to borrow books.

new_loan.html - Register New Loans

The New Loan page streamlines the process of lending books:

  • Loan Form: Users can select a customer, choose a book, and specify the loan period.
  • Backend Integration: The form communicates with the backend to record the loan and update the book availability in real-time.

return.html - Return Books

This page allows users to return borrowed books and update the system accordingly:

  • Return Form: Simple and easy-to-use, allowing users to mark a book as returned, automatically removing it from the active loans list.
